# 🌐 Portfolio Website

A modern Full Stack Portfolio Website with a secure Admin Dashboard to manage portfolio content dynamically.

## 🚀 Features

- Responsive Design
- Admin Dashboard
- Secure Authentication
- Dynamic Content Management
- Resume & Image Upload
- Contact & Social Links

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 📂 Project Structure

```
PortfolioWebsite/
├── client/
├── server/
└── public/
```
