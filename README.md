# 🌐 Portfolio Website

A modern Full Stack Portfolio Website with a secure Admin Dashboard to manage portfolio content dynamically.

## 🚀 Features

- Responsive Design
- Secure Authentication
- Admin Dashboard
- Dynamic Portfolio Management
- Skills, Projects, Experience & Education CRUD
- Resume & Profile Image Upload
- Contact & Social Links Management

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- HTML5
- CSS3
- JavaScript (ES6+)
- Axios

### Backend
- Node.js
- Express.js
- JWT Authentication
- Multer
- Bcrypt.js

### Database
- MongoDB Atlas
- Mongoose

### Tools & Technologies
- Git
- GitHub
- VS Code
- REST API
- Postman
